# Example echo dapp

The following project holds an example DAPP to be used for dev purposes.

## Installation

```bash
npm install
```

## Compiling the smart contracts

```bash
npm run compile
```

## Development
In one terminal execute:

```bash
npm run dev-blockchain
```

and in the other

```bash
npm run deploy-local
```

## Unit tests

```bash
npm run test
```
